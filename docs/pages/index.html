Quick Examples

Run from the command line. See full CLI documentation →

Download a model and ask it a question.

$ talu ask -m Qwen/Qwen3-0.6B "What is 2+2?"

Use environment variables for quick iteration.

$ TOKENS=100 TEMPERATURE=0.7 talu ask -m Qwen/Qwen3-0.6B "Tell me a joke"

Quantize a model for faster inference.

$ talu convert -m Qwen/Qwen3-0.6B --scheme gaf4_32
