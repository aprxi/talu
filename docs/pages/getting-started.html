▀█▀talu

    Getting Started

    High-performance LLM inference with zero dependencies

    Install

    $ pip install talu

    Python 3.10+ · PyPI · Releases · Build from source

    Quick Examples

    From the command line. CLI Guide →

    Download a model and ask it a question.

    $ talu get LiquidAI/LFM2-350M
    $ talu ask -m LiquidAI/LFM2-350M "What is 2+2?"

    Set a default model, then use it without -m.

    $ talu set LiquidAI/LFM2-350M
    $ talu ask "Tell me a joke"

    Quantize a model to 4-bit for faster inference.

    $ talu convert -m LiquidAI/LFM2-350M --scheme gaf4_32

    Python

    Same models, from Python. Python Examples →

    Chat.

    from talu import Chat

chat = Chat("LiquidAI/LFM2-350M")
response = chat("What is the capital of France?")
print(response)

    Multi-turn conversation.

    from talu import Chat

chat = Chat("LiquidAI/LFM2-350M", system="Be concise.")
response = chat("What is the largest planet?")
response = response.append("How far is it from the Sun?")
print(response)
