    Usage

    From HuggingFace Hub

    import talu

# Downloaded automatically on first use
response = talu.generate("Qwen/Qwen3-0.6B", "Hello!")

    From Local Path

    response = talu.generate("/path/to/model", "Hello!")

    Private Models

    Set the HF_TOKEN environment variable:

    export HF_TOKEN=your_token_here
    response = talu.generate("your-org/private-model", "Hello!")
